\Section{Activités expérimentales}

Nous allons étudier le montage 2 dont le schéma de principe est
présenté dans la partie théorique.

\vressort{1}

On prendra $R_1 = 1~k\Ohm$ et $R_2 = 4,7~k\Ohm$.

\vressort{1}

Nous souhaitons tracer expérimentalement et point par point la
caractéristique Entrée - Sortie de ce montage ($V_s$ en fonction de
$V_e$).


\vressort{3}

\begin{enumerate}
\item Indiquez le matériel nécessaire pour réaliser ce montage.\\
Faites vérifier votre liste de matériel.

\vressort{1}

\item Réalisez le schéma du \emph{montage} permettant de remplir l'objectif. On rappelle notamment que doivent figurer sur un schéma de montage les différentes alimentations (hormis l'alimentation symétrique) et les appareils de mesure.\\
Faites vérifier votre schéma de montage.

\vressort{1}

\item Réalisez le montage sans mettre sous tension.\\
Faites vérifier le montage lorsque vous pensez que tout est prêt (y
compris les appareils de mesure)
\begin{itemize}
\item Résistances
\item Fil entre $E^+$ et Masse
\item Tension d'entrée
\item Alimentation symétrique
\item Instruments de mesure
\item Propreté du montage (respect des couleurs conventionnelles, fil non
mélés, ...)
\end{itemize}

\vressort{1}

\item Tracez $V_s$  en fonction de $V_e$ pour $V_e$ allant de $-4~V$ à $+4~V$ tous les $0,5~V$.\\
Vous utiliserez une feuille de papier millimétré voir simplement une
feuille quadrillée à $0,5~mm$ pour effectuer ce tracé. Le choix de
l'échelle est libre mais doit figurer sur le tracé.

\vressort{1}

\item Calculez le coefficient directeur de la droite et déduisez-en la
  relation entre $V_s$ et $V_e$.\\
Comparez avec la valeur théorique.

\vressort{1}

\item Comment tracer directement à l'oscilloscope $V_s$ en fonction de $V_e$ ?\\
Faites un schéma de montage et expliquez l'alimentation que vous utilisez, le mode dans lequel vous faites fonctionner l'oscilloscope...

\end{enumerate}
