\Section{La lumière est une onde}

\frame{
\frametitle{\insertsectionhead}
\pause

La lumière est une onde électromagnétique à laquelle notre oeil est sensible.

\pause

L'émission de lumière est due à la vibration des électrons périphériques des atomes qui constituent la source.
}

\frame{

\`A toute onde électromagnétique correspond :
\pause
    \begin{itemize}
    \item dans le temps : une période T (période temporelle)
\pause
    \item dans l'espace : une longueur d'onde $\lambda$ (période spatiale)
      \begin{itemize}
\pause
      \item[] $\lambda$ est la distance parcourue par l'onde pendant la période temporelle $T$
      \end{itemize}
    \end{itemize}
}

\frame{
  \begin{itemize}
  \item Une lumière monochromatique correspond à une vibration à une
    fréquence unique.
    \pause
  \item En général, la lumière est la superposition de différentes
    longueur d'onde.
    \pause
  \item La lumière blanche peut être décomposée par un prisme pour
    former un spectre.
    \pause
  \item Le spectre visible est le domaine de radiation pouvant être
    captées par notre oeil.\\
    \pause
La longueur d'onde dans le vide $\lambda_0$ doit être comprise entre
$400~nm$ (violet) et $800~nm$ (rouge) pour être visible par l'oeil
humain.\\
\pause
Au dessus de $800~nm$ c'est le domaine de l'infra-rouge.\\
\pause
En dessous de $400~nm$ c'est le domaine de l'ultra-violet.
  \end{itemize} 
}

\frame{

  La vitesse de propagation dépend du milieu traversée.\\

\pause

  Celui-ci est caractérisé par son indice de réfraction noté $n$.

\pause

  \[n=\frac{c_0}{c}\]

\pause

\begin{unites}
\item  $c_0$ : vitesse de propagation (célérité) de la lumière dans le vide
  ($c_0 \approx 3.10^8~m/s$)

\pause

\item $c$ : vitesse de propagation de la lumière dans le milieu de
  propagation (en $m/s$)
\end{unites}

\pause

  \emph{Remarque :} $n$ est toujours supérieur à $1$ car la vitesse de propagation dans un milieu quelconque est toujours inférieure à la vitesse de propagation dans le vide.

}


\frame{

  La longueur d'onde de la lumière $\lambda$ (dans un milieu donné) est définie par :

\pause

  \[\lambda = c T = \frac{c}{\nu}\]

\pause

\begin{unites}
\item  $c$ : vitesse de propagation de la lumière dans le milieu de
  propagation (en $m/s$)
\pause
\item  $T$ : période temporelle $T$ (en $s$)
\pause
\item  $\nu$ : fréquence de l'onde (en $Hz$)
\end{unites}

\pause

\[\nu = \frac{1}{T}\]

}

\Subsection{Expérience}

\frame{
\frametitle{\insertsubsectionhead}
\pause


    On éclaire une plaque de zinc avec une lumière violette.

\pause
  
    Des électrons sont arrachés à la plaque grâce à l'énergie cédée par la lumière.

\pause

    Ces électrons échappent à l'emprise des atomes de zinc.

\pause


    La même expérience avec une lumière rouge ne produit aucun effet
    (même si on prolonge l'éclairage ou si l'on éclaire plus
    intensément)


}

\Subsection{Interprétations}

\frame{
\frametitle{\insertsubsectionhead}
\pause
    
    La lumière ne lâche pas son énergie de manière continue mais par paquets.
  
\pause

    Ces paquets d'énergie sont plus au moins importants selon les couleurs.
    
\pause
    
    L'atome ne laisse partir un électron que si le paquet reçu possède
    une certaine valeur minimale.

\pause
    
    La lumière rouge ne possède pas cette valeur minimale d'énergie
    (alors que la lumière violette la possède)

\pause
    
    On peut dire que la lumière se comporte comme un flux de
    particules infiniment petites, sans masse ni charge appelées
    \emph{photons}.

}


\frame{
  \`A une lumière monochromatique de fréquence $\nu$ et de longueur
  d'onde $\lambda = \frac{c}{\nu}$,

\pause

  on associe des photons d'énergie $E$ telle que :

\pause

  \[E = h \nu\]

\pause

\begin{unites}
\item  $h$ : constante de Planck $h=6,626.10^{-34}J.s$

\pause

\item  $E$ : énergie en Joule ($J$) ou parfois en électron-volt\\
 $1~eV = 1,6.10^{-19}~J$
\end{unites}

}
