\emph{Remarque : }
Pour le réglage du collimateur, vous devez observer les \emph{bords} de la fente source pour faire le réglage.




\section{Mesure de l'angle du prisme}
\emph{Remarques : }
\begin{itemize}
\item Travaillez avec une fente fine
\item Utilisez la vis de réglage fin pour la position de la lunette
\end{itemize}


Estimez l'incertitude $\Delta A$ sur la détermination de l'angle $A$ du prisme.





\section{Mesure de l'angle de déviation minimale pour une longueur
d'onde donnée}

La source est une lampe à vapeur de mercure.


Les longueurs d'onde des différentes raies vous sont données en annexe.


Vous ferez la mesure de $D_m$, angle de déviation minimale, pour les radiations les plus intenses et sur le domaine de longueur d'onde le plus large possible.

\subsection{Mesure d'une longueur d'onde inconnue}

\subsubsection{Courbe d'étalonnage du prisme}
Tracez sur papier millimétré $D_m = f(\lambda)$ pour les radiations de la lampe à vapeur de mercure.

\subsubsection{Mesure d'une longueur d'onde inconnue émise par le
  sodium}

\begin{itemize}

\item Remplacez la lampe à vapeur de mercure par la lampe à vapeur de
sodium.


\item Mesurez $D_m$ pour le doublet jaune du sodium.


\item En déduire grâce à la coube d'étalonnage du prisme la longueur d'onde correspondante.

\end{itemize}
