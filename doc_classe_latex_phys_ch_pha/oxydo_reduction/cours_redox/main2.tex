\section{Expérience préliminaire}
\subsection{Expérience}
Dans un tube à essai, introduire une spatule de fer en poudre. Ajouter environ $5~mL$ de sulfate de cuivre ($Cu^{2+}$, $SO_4^{2-}$) à la concentration $c = 0,1~mol.L^{-1}$.


\subsection{Observations}
\begin{itemize}
\item \trouligne{dépôt de solide rouge sur le fer}
\item \trouligne{la coloration bleu de la solution devient plus pâle}
\end{itemize}

\subsection{Interprétations}
\begin{itemize}
\item \trouligne{Le cuivre présent dans la solution s'est déposé sur le fer.}
\item \trouligne{les ions $Fe^{2+}$ sont apparus.}
\end{itemize}


\subsection{\'Equation bilan} % Demi-équations et 
\trouformule{%
Cu^{2+} + Fe \rightarrow Cu + Fe^{2+}
}

L'équation bilan s'explique par un transfert d'électron :

\begin{itemize}
\item \trouligne{l'ion $Cu^{2+}$ doit prendre 2 électrons se transformer en atome de $Cu$}.
\item \trouligne{le fer doit libérer 2 électrons pour se transformer en ion fer II $Fe^{2+}$}.
\end{itemize}

\'Ecrivons les deux demi-équations électroniques.

\begin{enumerate}
\item \trouligne{$Cu^{2+} + 2 e^- \rightarrow Cu$}
\item \trouligne{$Fe \rightarrow Fe^{2+} + 2 e^-$}
\end{enumerate}

\section{Couples rédox}

L'oxydant (\trou{$Cu^{2+}$}) et le réducteur (\trou{$Cu$}) constituent un \trou{couple rédox}. On note toujours le couple $ox/red$ (l'oxydant en premier). Il s'agit donc ici du couple \trou{$Cu^{2+}/Cu$}.


L'autre couple rédox rencontré dans l'expérience préliminaire est \trou{$Fe^{2+}/Fe$}.


Autres exemples de couples rédox : \trou{$Ag^{+}/Ag$}, \trou{$Al^{3+}/Al$}, \trou{$Mg^{2+}/Mg$}, ...

\remarque{
Dans un couple rédox comportant un métal, le métal est \trou{le réducteur} et le cation métallique \trou{l'oxydant}.
}

\subsection{Conclusions}
\subsubsection{Première conclusion}
\begin{itemize}
\item \trouligne{Le zinc réduit tout. C'est le réducteur le plus fort de l'expérience.}
\item \trouligne{L'ion $Ag^+$ oxyde tout.}
\end{itemize}

\pagebreak

\subsubsection{Deuxième conclusion}
\begin{itemize}
\item \trouligne{2 expèces d'un même couple rédox ($Cu^{2+}$ et $Cu$ par exemple) ne réagissent pas ensemble}.
\item \trouligne{Si la réaction a lieu dans un sens, elle n'a pas lieu dans l'autre sens.}\\
	 \trouligne{Le sens dépend de la force de l'oxydant et du réducteur qui réagissent ensemble}
\end{itemize}


\subsubsection{Conclusion générale : règle du $\gamma$ (gamma)}


\begin{center}
Echelle potentiels rédox
\vspace{8cm}
\end{center}

L'oxydant le plus fort réagit avec le réducteur le fort : c'est la règle dite du $\gamma$ (\trou{gamma}) (à cause de la forme de la lettre).

Ici on voit que \trou{$Fe$} et \trou{$Cu^{2+}$} réagissent ensemble mais que \trou{$Fe^{2+}$} et \trou{$Cu$} ne réagissent pas ensemble.

\section{Place de l'hydrogène dans la classification électrochimique}

\subsection{Action de l'acide chlorhydrique sur le fer} %sur le cuivre
\subsubsection{Expérience}
Dans le tube contenant du fer, ajoutons de l'acide chlorhydrique.

\subsubsection{Observations}
\begin{itemize}
\item \trouligne{Un dégagement gazeux se produit, c'est du dihydrogène $H_2$.}
\item \trouligne{La solution prend une couleur verte caractéristique des ions fer II $Fe^{2+}$.}
\end{itemize}

\subsubsection{Demi-équations et équation bilan}

Ecrire les deux demi-équations.

%\begin{itemize}
%\item
\trouformule{%
Fe_{(s)} = Fe^{2+}_{(aq)} + 2e^-
}

%\item

\trouformule{%
2 H^+_{(aq)} + 2 e^- = {H_2}_{(g)}
}

%\end{itemize}

En ajoutant les deux demi-équation memebre à membre, on obtient :

\trouformule{%
Fe_{(s)} + 2 H^+_{(aq)} + 2 e^- \rightarrow Fe^{2+}_{(aq)} + 2e^- + {H_2}_{(g)}
}

En éliminant les électrons, on a :

\trouformule{%
Fe_{(s)} + 2 H^+_{(aq)} \rightarrow Fe^{2+}_{(aq)} + {H_2}_{(g)}
}

Les ions $Cl^-$ n'ont pas réagi. On dit qu'ils sont spectateurs.

\trouformule{%
Fe_{(s)} + 2 H^+_{(aq)} + Cl^- \rightarrow Fe^{2+}_{(aq)} + {H_2}_{(g)} + Cl^-
}

\subsection{Action de l'acide chlorhydrique sur le cuivre} 
\subsubsection{Expérience}
Dans le tube contenant du fer, ajoutons de l'acide chlorhydrique.

\subsubsection{Observations}
\trouligne{%
On n'observe pas de réaction.
}

\subsection{Place du couple ${H^+}_{(aq)} / {H_2}_{(g)}$}

D'après ce qui précède, nous en déduisons que \trou{$H^+$ est plus oxydant que $Fe^{2+}$} mais moins que \trou{$Cu^{2+}$}. Nous pouvons donc positionner le couple ${H^+}_{(aq)} / {H_2}_{(g)}$ par rapport aux autres couples étudiés (\trou{entre ces deux couples}).

Le positionnement peut être affiné en remarquant que quelques billes de plomb sont également (comme le fer) attaquées par de l'acide chlorhydrique concentré. Il se produit un dégagement gazeux de dihydrogène. $H^+$ est donc plus \trou{oxydant} que $Pb^{2+}$.
