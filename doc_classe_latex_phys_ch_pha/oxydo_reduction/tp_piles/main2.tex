\subsection{Réalisation des piles}

\begin{enumerate}

\item Réaliser les trois demi-piles :
  \begin{itemize}
  \item $Fe^{2+} / Fe$
  \item $Pb^{2+} / Pb$
  \item $Zn^{2+} / Zn$
  \end{itemize}
  ($c = 0,1~mol.L^{-1}$).

\item Associer chacunes d'elles à la demi-pile $Cu^{2+} / Cu$.

\item Déterminer la polarité de chacune des piles réalisées.

\item Mesurer la force électromotrice de chacune des piles.

\item Préciser pour chaque pile les demi-équations électroniques et
l'équation de fonctionnement.

\item \emph{Remarque :} bien décaper les lames métalliques avant
chaque utilisation.
